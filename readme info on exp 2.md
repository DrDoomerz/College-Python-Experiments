## Theory

### List Data Type in Python

A **list** in Python is an ordered, mutable collection of elements. Lists can store elements of **different data types** such as integers, floats, strings, and booleans. Lists are written using square brackets `[]` and elements are separated by commas.

**Key characteristics of lists:**

* Ordered
* Mutable (elements can be changed)
* Allows duplicate values
* Can store mixed data types

Example:

```python
list1 = [1, 2.5, "rachit", True]
```
